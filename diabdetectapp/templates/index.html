About This Project

This project focuses on predicting the likelihood of diabetes using patient medical data and basic machine learning concepts.

Users can register, log in, and submit medical details which are later used for prediction and analysis.

How It Works

  • User creates an account and logs in
  • Medical details such as glucose and BMI are entered
  • Data is processed using a prediction model
  • The system shows diabetes risk
